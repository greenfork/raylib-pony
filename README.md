# Raylib-Pony

[Raylib] bindings for [Pony].

[Raylib]: https://www.raylib.com/
[Pony]: https://www.ponylang.io/

## Develop

- Install Pony
- Install [corral](https://github.com/ponylang/corral)

```shell
$ git clone --recurse-submodules https://github.com/greenfork/raylib-pony.git
$ cd raylib-pony
$ corral fetch
```
